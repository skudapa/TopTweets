GenerateLinks.java

GenerateLinks.java is a twitter client program that searches the 100 most recent tweets with links, for a supplied hashtag. 
The script ignores links to hashtags and twitter profiles and tries to print only user entered links.

Requirements
Required JDK 1.6.X and higher, Tested and developed in JDK 1.6.X and higer.For more info Visit http://www.oracle.com/technetwork/java/index.html


Usage
Run the script by passing a single alphanumeric hashtag as the argument. eg. java GenerateLinks radiohead
